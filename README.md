# 💰 Expense Tracker

A sleek, modern, and minimalistic web application designed to track and manage your daily financial transactions with full bilingual support (English & Arabic).

## 🌟 Features

- **Full Localization (Bi-lingual):** Toggle between English and Arabic seamlessly with a single click. The layout dynamically switches directions (LTR/RTL) along with currency labels (EGP / جنيه).
- **Minimalistic & Modern UI:** A clean interface built with subtle animations, card layouts, and custom interactive form controls that are easy on the eyes.
- **Real-time Analytics:** Automatically calculates and updates your Total Balance, Total Income, and Total Expenses instantly as you add or delete transactions.
- **Interactive Type Selector:** Upgraded radio buttons that transform into fully styled clickable cards instead of the basic browser default inputs.
- **Fully Responsive:** Tailored layout using CSS Grid and Flexbox to ensure an optimal user experience across mobile phones, tablets, and desktops.

---

## 🛠️ Technologies Used

- **HTML5:** Semantic architecture for accessible and clean markup.
- **CSS3 (Custom Variables & Grid):** Clean styles utilizing CSS variables for consistent theming and adaptive grids for layout layouts.
- **Vanilla JavaScript:** Clean DOM manipulation, array processing (`forEach`, `filter`), and logic handling for localization and computations.
